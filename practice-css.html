Selectors, Properties & Values

CSS allows you to style HTML elements by selecting them and applying style rules.

  • Selectors: Choose elements to style (e.g., `.class`, `#id`, `tag`).
  • Properties: Define what to style (e.g., `color`, `background`, `margin`).
  • Values: Set how it looks (e.g., `red`, `20px`, `flex`).
/* CSS */
.box {
  background: teal;
  color: white;
  padding: 1rem;
  border-radius: 6px;
}
<!-- HTML -->
<div class="box">Styled Box</div>
